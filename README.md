# sports-analytics-lab
Sports Analytics Lab is a fully end-to-end applied analytics project combining exploratory data analysis, predictive modelling, and interactive BI dashboards. The objective is to demonstrate how data science, football statistics, and machine learning can be used to evaluate historical performance and forecast future outcomes using real-world data.
